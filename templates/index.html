About

What is Ablaze?
Ablaze (formerly DihFind) is a search engine and website proxy service built for vintage browsers. It rewrites modern sites for simpler, text-friendly viewing by stripping out JS, modern CSS and HTML. Sites remain somewhat functional. Too modern sites that rely heavily on JS or modern CSS may not work properly.

Why so few results?
The system limits responses to the first 50 matches for speed and clarity. Actually, you will get about more or less 35 results at a time.

How does it work?
In our current infrastructure, a Python Flask webserver on PythonAnywhere that allows HTTP gets your request. It then forwards it to our Ablaze server on Vercel which enforces HTTPS. For searches, it uses the Python ddgs module and sends back the response. For proxying sites, it requests that specific site (note that all sites don't work), gets its source code, strips out modern HTML/CSS/JS and returns the cleaned page back.

What happens to my browsing data?
Your data is never shared outside Ablaze. Your site data (cookies only) are completely wiped out from our DB after 1 hour of inactivity and NEVER RESTORED. This also means that not using Ablaze for 1 hour will completely reset your login info for websites, etc.

Why was Ablaze made?
To keep old machines exploring the modern Internet without breaking, like IE 5.0 on Windows 98 SE, as well as old Mac's.

What was Ablaze inspired by?
Ablaze was inspired by FrogFind, made by Action Retro (YouTuber).
